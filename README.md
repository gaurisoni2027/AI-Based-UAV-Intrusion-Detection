# AI-Based Intrusion Detection for UAV Communication Networks Using Machine Learning and Deep Learning

## Project Description
This project aims to develop Machine Learning and Deep Learning based Intrusion Detection Systems (IDS) for securing UAV communication networks against cyberattacks.

## Dataset
Cyber-Physical Dataset for UAVs Under Normal Operations and Cyberattacks (IEEE T-ITS Dataset)

## Attacks Considered
- DoS Attack
- Replay Attack
- Evil Twin Attack
- False Data Injection Attack
- Benign Traffic

## Objectives
1. Perform Exploratory Data Analysis (EDA).
2. Build baseline ML models.
3. Build Deep Learning models.
4. Compare model performance.
5. Develop a research-oriented methodology suitable for publication.
